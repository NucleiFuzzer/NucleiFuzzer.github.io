About

NucleiFuzzer is an advanced automation tool that streamlines web application security testing by combining the power of Nuclei, ParamSpider, Waybackurls, Gauplus, Hakrawler, and Katana. It efficiently discovers hidden endpoints, parameters, and URLs, then scans them with fuzzing templates to uncover vulnerabilities with speed and accuracy.

Designed for security professionals, bug bounty hunters, and developers, NucleiFuzzer ensures robust protection by integrating URL validation, deduplication with uro, and HTTP filtering using httpx, providing a seamless workflow for vulnerability detection and analysis.







Comprehensive Discovery

Integrates multiple tools to uncover URLs, parameters, and endpoints missed by others.

Enhanced Scanning

Uses Nuclei with fuzzing templates to detect a wide range of vulnerabilities precisely.

Smart Filtering

Employs uro and httpx for deduplication and HTTP validation to focus only on meaningful results.

Optimized Performance

Supports rate limiting and batch scans for fast, efficient, and customizable workflows.
